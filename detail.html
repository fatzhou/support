Other Android 7.x Guide

  • Category Other, Android 7

Essential Settings

These settings must be properly configured to keep iKeyMonitor working.

Battery Settings

  1. Settings > Battery > Battery saver (1, 2, 3)
    Set to “OFF”

  2. Settings > Battery > 3-Dot menu > Battery optimization > All apps > Internet Service (1, 2, 3)
    Set to “Don’t optimize”

Background App Setting

No special configuration found.
You may Ignore this step.

Other Settings

No need to change these settings by default. But they will affect iKeyMonitor functions if changed by user.

Data Usage

  1. Settings > Data usage > Data saver > Unrestricted data access > Internet Service (1, 2, 3)
    Set to “ON”

  2. Settings > Data usage > Cellular data usage > Internet Service > App data usage – Background data (1, 2, 3)
    Set to “ON”

  3. Settings > Data usage > Wi-Fi data usage > Internet Service > App data usage – Background data & Unrestricted data usage (1, 2, 3)
    Set to “ON”

App Permission

Settings > Apps > Internet Service > App info > Permissions > App permissions (1, 2, 3)
Allow all permissions
